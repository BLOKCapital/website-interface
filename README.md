# BLOK Capital — Website

Marketing site for BLOK Capital, a non-custodial on-chain wealth management protocol.

> "It's Crypto But Different."

## Pages

- **Home** — hero, how it works, the garden metaphor, social proof, roadmap, and DAO community
- **Features** — what the product offers for investors, managers, and builders
- **About** — story, core values, team, and governance
- **Preview** — product walkthrough and interactive sandbox
- **Contact** — channels, FAQ, and legal info

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Built with

Next.js, React, TypeScript, Tailwind CSS, and Framer Motion.
